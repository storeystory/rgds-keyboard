A keyboard for the Anbernic RG DS. It populates the entire bottom screen, and is navigated via the built-in controls.

- D-pad or left stick for changing what character you have selected.
- A button or R1 to send the currently selected character.
- X button to add a space. Quickly double-tap for a period and a space.
- B button or L1 to backspace. Press B at the beginning of a text field to close the keyboard.
- Y button to toggle case, or to return to letters from symbols.
- L2 to toggle between selector navigation and cursor navigation (changing whether you're moving the text cursor within the text field or moving the character selector within the keyboard)
- R2 to toggle caps lock
- L3 to enter
- R3 to toggle between symbols and letters
- Customize the vibration intensity, colors, and background image in the built-in settings app\
